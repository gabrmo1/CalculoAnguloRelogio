# Objetivo

Esse programa tem o objetivo de calcular em ângulos a relação entre dois ponteiros de relógio, levando em consideração a variação do ponteiro menor(das horas) em relação à posição do ponteiro maior(dos minutos).

## Como usar

O programa solicita os dados(horas e minutos) e então calcula a relação de distância entre os ponteiros do relógio em ângulos.

## Dados de desenvolvimento
O programa foi desenvolvido em Java(JDK8) Pela IDE Eclipse(2020/06) no Windows 10 Home 64 bits.
